 Поведение пользовательского анализа в приложении Sleepy

Кейс по продуктовой аналитике

Бизнес задача.
Sleepy — приложение для улучшения качества сна.
Монетизация — подписка 990 ₽ / месяц после 14-дневного trial.

Стоимость привлечения пользователя — 500 ₽.

Цель анализа:
- оценить retention пользователей
- определить точку окупаемости
- рассчитать LTV продукта

Метрики

- DAU
- MAU
- Stickiness
- Retention
- Lifetime
- LTV

Результаты

- retention падает на 18 п.п. после paywall
- CAC окупается на 16 день
- средний lifetime — 6 месяцев
- LTV = 5940 ₽
